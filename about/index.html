About Me

I am a developer and musician from Vancouver, BC, and can be found as @iKevinY on most places online. I enjoy programming, composing music, and playing the piano. My avatar is a lilac-breasted roller.

This website is generated with Pelican, a static site generator written in Python. It is hosted on GitHub Pages and the source can be found in this GitHub repository. Its theme is Pneumatic, a modified version of a theme created by Greg Reda, which itself is built using Skeleton. The syntax highlighting in code blocks uses Chris Kempson’s Eighties Dark theme. Articles on this site are licensed under a CC BY 4.0 license.

These are some of the applications, websites, and technologies that I use on a regular basis:

  • Sublime Text — source code editor with a wide variety of plugins available
  • GitHub — online Git-based project hosting
  • TotalTerminal — hotkey-activatable terminal overlay
  • f.lux — adjusts the colour temperature of displays at night to reduce eye strain
  • Rainwave — video game music Internet radio station
  • Radium — menu bar application for streaming Internet radio stations

Here some books that I highly recommend reading:

  • Code — Charles Petzold explains the design of computers and software in a manner accessible to even those without a background in computer science.
  • The Signal and the Noise — Nate Silver discusses how mathematical models created using probability and statistics are applied to real-world situations.
